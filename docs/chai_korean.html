Chai & HanGyr

  • Chai 자모는 4점1선 모형의 공통한 구조에서 산생된 규칙 있는 글자 부호이다.
  • Chai 자모는 정방형의 4개 끝점중 왼쪽우에서 부터 시작되여 파생한 부호 30개중 기억하기 쉽게 26개을 형태가 영문자모와 비슷한 부호을 자모로 설정하엿다.
  • Chai 수자는 정방형의 4개 끝점중 오른쪽위에서 부터 시작되여 파생한 부호중 10개을 수자로 설정하엿다.
  • Chai 콘트롤 부호 와 기타 부호는 왼쪽아래 와 오른쪽아래에서 부터 시작되는 파생부호을 사용한다.
  • Chai 자모 발음은 한글 발음 규칙을 기본적으로 준수하며 자모와 발음이 상대적으로 고정적이다.
  • Chai: {d, w, t}, {r, n, h}, {m, l}, {f(*), v, b, p}, {g, q, k}, {z, j, c}, {s, x}.
  • 한글 자음: {ㄷ, ㄸ, ㅌ}, {ㄹ, ㄴ, ㅎ}, {ㅁ, ㅇ}, { f(*한글중 F 발음없음), ㅂ, ㅃ, ㅍ}, {ㄱ, ㄲ, ㅋ}, {ㅈ, ㅉ, ㅊ}, {ㅅ, ㅆ} Chai 자음 발음에 대응 된다.
  • Chai {y, i, e, a, o, u},{yi, ye, ya, yo, yu},{è=ie, à=ia, ò=io, ù=iu},{oa, oà, ue, uè},{yè, yà}.
  • 한글 모음: {ㅡ, ㅣ, ㅓ, ㅏ, ㅗ, ㅜ} {ㅢ, ㅕ, ㅑ, ㅛ, ㅠ},{ㅔ, ㅐ, ㅚ, ㅟ},{ㅘ, ㅙ, ㅝ, ㅞ},{ㅖ,ㅒ} Chai 모음 발음에 대응 된다.
  • Chai: {ai, ei, oi, ui},{èi,òi,ùi},{oai,uei,uiei},{yèi,yài},{au,eu,ou} .
  • Chai 기타 조합모음:{아이,어이,오이,우이},{에이,외이,위이},{와이,워이,웨이},{예이,얘이},{아우,어우,오우} 대응 되게 발음 된다.
  • Chai: {ia=à},{ie=è},{io=ò},{iu=ù}.
  • Chai 모음 간략 자모:{ㅐ= à},{ㅔ= è},{ㅚ= ò},{ㅟ= ù} 로 대체 하여 (예: èi=에이, yà=얘) 사용에 편리을 준다.
  • Chai: X@{d,w,t,r,n,h,m,l,f,v,b,p,g,q,k,z,j,c},Y@{y,i,e,a,o,yi,ye...} => (X+Y),(X+Y+X),(XX+Y),(XX+Y+X), {da,tai,dol,tan,stop}
  • Chai 글자조합은 (자음+모음),(자음+모음+자음,자음+자음+모음),(자음+자음+모음+자음) 조합 형식을 택한다:{다= da,타이= tai, 동= dol,탄= tan,스톺= stop}
  • Chai 글자조합(자음+자음+모음) [ㅅㅌㅗㅍ]=[스톺]= { stop=sytop} 첫번째 자음은 기본 자음으로 발음한다.
  • Chai 글자조합[자음+모음+자음+모음]경우 뒤에 모음 우선원칙으로[(자음+모음)+(자음+모음)] { piny=(pi)(ny)}으로 분리 하여 발음한다.
  • Chai 단어발음 표기 중 매개 글자 첫번째 자음을 대문자로 표기하면 { hangyr=HanGyr=한글} 식별에 도움이 된다.
  • Chai 단어 { ZulLal}=중앙, { oal=Loal}=왕, 첫번째 { L} 는 모음 앞에 붙여 발음 하지 안고 단어 조합시 식별을 위해서다.
