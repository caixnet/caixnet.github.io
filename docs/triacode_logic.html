COMPUTER SYSTEM CAPABLE OF PERFORMING HYBRID OPERATION OF TERNARY OPERATION AND BINARY OPERATION

  • Disclosed is a computer system capable of performing a binary operation and a ternary operation in parallel and switching between the binary operation and the ternary operation.
  • A hardware device of the computer system comprises a processor, a storage system, an encoding system, an input system and a network system capable of switching between the binary operation and the ternary operation.
  • The hardware device further comprises an embedded virtual machine management firmware; the virtual machine management firmware generates a virtual machine in a binary computing processing mode and/or a virtual machine in a ternary computing processing mode via the configuration of the computer system when the computer system starts, and provides corresponding binary system and/or ternary system computing processing at the generated binary virtual machine and/or the ternary virtual machine, thus being suitable for both electronic computing processing and photonic computing processing.
