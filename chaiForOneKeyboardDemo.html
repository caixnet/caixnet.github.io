Chai ForOne(Four points One string) keyboard IME Demo

  • Chai ForOne is a keybaord that base on chaicode solution.
  • Chai ForOne kyeboard codes have basic and customise option.
  • Chai ForOne basic codes reserved and observe chaicode.
  • Chai ForOne customise codes can flexible define ForOne codes.
  • Click Chai ForOne keyboard to click active and background color changed.
  • You can use mouse move each 4 points and click mouse end swipe.
  • Double Click left-top point to change shift-mode.
  • Double Click right-top point to "delete" key.
  • Double Click right-bottom point to "enter" key.
  • Double Click left-bottom point to "option" key to change setting.
  • Define points: left-top=1,right-top=2,right-bottom=3,left-bottom=4.
  • Chai ForOne keyboard supported below keys, others define customises.
