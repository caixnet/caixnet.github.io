Chia & LilGyRiSiu

  • All Chia characters are from one structure on .
  • One symbol in Chia is made up of 4 endpoints from the corners of a square and 1 stroke to connect the 4 endpoints.
  • Chia symbols starting from upper left corner are desigined for 26 alphabets.
  • Chia symbols starting from upper right corner are desigined for 10 numbers.
  • Chia alphabet have 28 symbols(2 symbols are reserved).
  • It's normal used 26 symbols: 6 base and 30 combine vowels, 20 consonants.
  • Chia is a phonetic alphabet generated from Chia symbols. Pronunciation and spelling can be one to one match or vowel combinations.

    Chia & Wearable

    • Wearable devices require simply and unified gesture coding for handling the complicated input and output. What would be the idea selection for encoding? It is suitable for human to grasp and at the same time easy for machines to recognize and process with low error rate.
    • When communicating, language and symbols are the most important tools to facilitate the interaction with each other and to pass down culture and thoughts through generations
    • Using one figure via one stroke to input the Chia symbols and it is processed to the respective localization characters, including alphabets, numbers, punctuations and control symbols.
    • Chia shift control symbols (|) is applied together with Chia alphabets and Chia numbers to generate the uppercase letters and punctuations.
    • When implementing Chia encoding in flat surface, tap the input box first; four endpoints appears in the screen. Four endpoints and two diagonals generate the four point shaped auxiliary frame. Users can use the auxiliary frame to motion input using Chia gesture symbols. When implementing Chia encoding in touch pad, four raised dots are designed in the touch pad. Users will use the four dots to facilitate to motion the Chia gesture symbols.
    • The uniqueness of Chia symbols makes it possible to motion Chia symbols on virtual screen in the air. User can use gesture recognition technology to remote control TV’s or other devices. Index finger is normally to motion the input and in certain conditions index finger and thumb can be used together to facilitate input and control the remote devices.
    • Chia symbol language is one of the line symbolic languages. Comparing to the other natural languages, Chia is a geometrical symbolic system which is more mathematic oriented. It is a simplified and geometric figure evolution of natural language symbols. Comparing to the typical dot symbolic language braille which is a binary encoding system, but Chia is a ternary encoding system. And Chia is also easier to learn and input comparing to Braille. A portable coin sized Chia keyboard with 3 or 4 touch endpoints can be designed. It can be used to replace the traditional 101 QWERT keyboard keys.
    • Chia can be used as a wearable device gesture language, with the flat surface input and 3D input to facilitate the information exchange. With the 4 endpoints routes and unified encoding rules, it is easy to learn and understanding. It can be used in different Chia adaptable devices.
    • Chia is a gesture symbol language designed for wearable devices. Chia gesture symbol language is a universal, simple, unified language which is used for man-machine interaction and is also easy to learn. Features of Chia make it an ideal solution for both PC and wearable devices. Furthermore, wearable devices are where Chia can show its advantage and promised future.
