Chia ForOne(Four points One string) keyboard IME Demo

  • Chia ForOne is a keybaord that base on chiacode solution.
  • Chia ForOne kyeboard codes have basic and customise option.
  • Chia ForOne basic codes reserved and observe chiacode.
  • Chia ForOne customise codes can flexible define ForOne codes.
  • Click Chia ForOne keyboard to click active and background color changed.
  • You can use mouse move each 4 points and click mouse end swipe.
  • Double Click left-top point to change shift-mode.
  • Double Click right-top point to "delete" key.
  • Double Click right-bottom point to "enter" key.
  • Double Click left-bottom point to "option" key to change setting.
  • Define points: left-top=1,right-top=2,right-bottom=3,left-bottom=4.
  • Chia ForOne keyboard supported below keys, others define customises.
  • 1234567890 1234567890, 0 have chiaForOneCode(21432, 23412).
  • abcdefghijklmnopqrstuvwxyz : abcdefghijklmnopqrstuvwxyz
  • ABCDEFGHIJKLMNOPQRSTUVWXYZ : ABCDEFGHIJKLMNOPQRSTUVWXYZ
  • ~`!@#$%^&*(){}[]-_=+\|/;:,.? '"<> and "space" use 12 ForOne code
  • chia41Code: ~4132 !32 @3143 #34123 ^314 *3142 (341 )432 {212 }121
  • [4321 ]3412 -21 _34 =43 +242 \31 |23 /42 ;141 :232 ,414 .323 ?321 '14 "41 <342 >431
  • Chia41code customise keys: "Hello,World!" = 41324 tab=423 your can define.
